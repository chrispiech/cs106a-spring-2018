Honor Code "Retractions":

for a vast majority of you, this announcement does not apply.

In CS106A we take the honor code very seriously. At the end of the class we will run a powerful piece of software which is very adept at detecting whether or not two submissions were generated by the same author (it is especially easy to detect if said author is someone who posted code on the internet). If you copied work from another person at any point in the course, please email Chris or head TA Julia. If you tell us about an honor code violation before it is discovered by our algorithm we will not send a case to the OCS.

Not sure if you crossed a line? We are also happy to help you figure out the shades of grey.
