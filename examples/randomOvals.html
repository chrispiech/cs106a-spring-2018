Random Ovals

Based on a handout by Eric Roberts

Topics: Javascript Graphics
Worked Example: Breakout Online!

Remember those good times making random circles with Java? Let's have those again, but in javascript! Write a program that has a canvas and two buttons, 'Make Circle!' and 'Clear Canvas'. When 'Make Circle!' is pressed, create a circle with a randomly chosen color and a random width and height, subject to the condition that the entire circle must fit inside the canvas without extending past the edge. One (of MANY) possible runs could look like the following:

To generate random numbers, use the javascript call Math.random(), which returns a number in between 0 and 1 (inclusive of 0 but not 1). How can you use this to generate random numbers in any range?

var randomNumber = Math.random(); // randomNumber has random value between 0 and 1

If you finish early, consider making random ovular doughnuts.
